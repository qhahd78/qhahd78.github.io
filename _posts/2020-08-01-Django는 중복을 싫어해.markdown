## URL Include
  *App 별로 URL을 관리할 수 있도록 구조화 
  *app 안에 직접 url을 만드는 것. 
  
  ```python
  
  from django.urls import path, include
  
  urlpatterns = [
    path('', include('page.urls')), 
    
    #page 안에 있는 url을 모두 불러와 project urls에서 관리한다. 
    # Url mapping 
] 

  ```
